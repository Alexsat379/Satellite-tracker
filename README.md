# 🛰️ Satellite Tracker

Visualizador en tiempo real de satélites (ISS) sobre la Tierra en 3D.

## 🌍 Demo

https://alexsat379.github.io/Satellite-tracker/

## 🚀 Características
- **Seguimiento en Tiempo Real:** Utiliza datos de latitud y longitud actualizados cada 2 segundos para posicionar la estación con precisión astronómica.
- **Entorno 3D Inmersivo:** Construido con Three.js, permitiendo rotar, hacer zoom y explorar la órbita libremente.
- **Modelo 3D Detallado:** Incorpora un modelo a escala de la Estación Espacial Internacional.
- **Iluminación Dinámica:** Simula la luz del sol para diferenciar el lado diurno y nocturno del planeta y cómo la luz rebota en la estación.

## 🛠️ Tecnologías utilizadas
- **Three.js:** Para la renderización y lógicas matemáticas del entorno 3D.
- **Vanilla JavaScript (ES6+):** Lógica del cliente y peticiones asíncronas.
- **HTML5 & CSS3:** Estructura e interfaz de usuario (etiquetas dinámicas CSS2D).

## 📚 Objetivo

Proyecto personal enfocado en visualización de datos espaciales y simulación en tiempo real

## 📜 Créditos y Agradecimientos

Este proyecto ha sido posible gracias a los recursos de dominio público y herramientas open-source:
