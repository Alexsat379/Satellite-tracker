# 🛰️ Satellite Tracker

Visualizador en tiempo real de satélites (ISS) sobre la Tierra en 3D.

## 🚀 Features
- Visualización 3D de la Tierra
- Posición en tiempo real de la ISS
- Datos obtenidos desde APIs públicas

## 🛠️ Tech Stack
- JavaScript
- Three.js
- Node.js

## 📦 Instalación

```bash
npm install

## ▶️ Uso

```bash
npm start
